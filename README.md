# Social Login Form

Um formulário de login moderno, minimalista e responsivo desenvolvido com HTML5 e CSS3. O projeto combina a autenticação tradicional por e-mail e senha com botões de login social estruturados para plataformas populares, envoltos em uma interface com forte contraste visual.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica dos blocos de formulário e botões de autenticação terceirizada.
* CSS3: Estilização de linhas de entrada personalizadas, gradientes lineares e gerenciamento de cores de marca.
* Flexbox: Alinhamento vertical dos componentes e centralização perfeita do card de login na tela.

## Funcionalidades e Técnicas Aplicadas

* Inputs com Linha Inferior (Minimalist Fields): Estilização de campos de texto sem bordas completas, mantendo apenas uma linha cinza sutil na base para entrada de dados limpa.
* Botão de Ação Principal em Gradiente: Botão "Continue" personalizado com um gradiente linear vibrante que transiciona entre tons de rosa e roxo escuro.
* Divisor de Seção Sutil: Inclusão de um separador textual clássico ("or Connect With Social Media") para segmentar as opções de autenticação.
* Botões de Login Social: Botões personalizados para o Twitter (azul ciano) e Facebook (azul escuro), integrando ícones de marca e alinhamento em blocos bem definidos.
